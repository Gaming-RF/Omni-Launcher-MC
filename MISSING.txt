## 🔴 Required Before First Use

### 1. CurseForge API Key
The CurseForge browsing feature requires an API key.  
Get one free at: https://console.curseforge.com/  
Add it in Settings > CurseForge API Key.

### 2. Microsoft Azure AD Client ID
`src-tauri/src/api/auth.rs` has `CLIENT_ID = "YOUR_CLIENT_ID_HERE"`.  
To enable Microsoft login you must register an Azure AD app:
1. Go to https://portal.azure.com/#blade/Microsoft_AAD_RegisteredApps/ApplicationsListBlade
2. Register a new app (Mobile and desktop application)
3. Set redirect URI to `http://localhost` (not needed for device code flow)
4. Copy the Application (client) ID into `auth.rs`

## 🟡 System Requirements

### 3. Linux Build Dependencies
On Ubuntu/Debian, install before building:
```
sudo apt install libwebkit2gtk-4.1-dev libsoup-3.0-dev \
  libjavascriptcoregtk-4.1-dev libssl-dev
```
Fedora: `webkit2gtk4.1-devel libsoup3-devel javascriptcoregtk4.1-devel openssl-devel`
Arch: `webkit2gtk-4.1 libsoup3 javascriptcoregtk-4.1 openssl`

### 4. Java 21+ (at Runtime)
Minecraft 1.20.5+ requires Java 21. The launcher auto-detects Java from:
- `JAVA_HOME` environment variable
- Common install paths (platform-specific)
- System `PATH`

You can also set a custom path in Settings > Java.

## 🟢 Nice-to-Have (Not Blocking)
